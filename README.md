## Prerequisiti

### Dal corso Data Analysis (30h)
- Python con Jupyter Notebook
- Pandas, numpy, matplotlib, seaborn
- Data cleaning e preprocessing

### Dal corso Fondamenti AI (35h)
- Storia e tipologie dell'AI
- Supervised/Unsupervised/Reinforcement Learning (concetti)
- Deep Learning: perceptron, MLP, CNN, RNN, Transformer (concetti)
