# Corso AI Generativa

Benvenuti al **Corso pratico di AI Generativa con Claude** - 40 ore di formazione intensiva organizzato da ITS Steve Jobs.

## Obiettivi del Corso

Questo corso ti guiderà attraverso i fondamenti dell'Intelligenza Artificiale Generativa, con focus pratico sull'utilizzo di Claude e delle API Anthropic.

## Struttura del Corso

Il corso è organizzato in **7 giornate** intensive:

| Giorno | Argomento | Descrizione |
|--------|-----------|-------------|
| **1** | [Intro AI Generativa](libro/giorno-01/index.md) | Cos'è l'AI Generativa e come funzionano gli LLM |
| **2** | [Prompt Engineering](libro/giorno-02/index.md) | Anatomia di un prompt e tecniche avanzate |
| **3** | [API Claude](libro/giorno-03/index.md) | Introduzione alle API Anthropic |
| **4** | [Claude Code](libro/giorno-04/index.md) | Sviluppo assistito con Claude Code |
| **5** | [Multimodalità](libro/giorno-05/index.md) | Modelli per immagini e input multimodali |
| **6** | [Avanzato ed Etica](libro/giorno-06/index.md) | Applicazioni avanzate, etica e limiti |
| **7** | [Ripasso e Progetti](libro/giorno-07/index.md) | Ripasso finale e presentazioni progetti |

## Come Usare Questo Materiale

Ogni giornata include:

- **Teoria**: Spiegazioni dettagliate dei concetti
- **Pratica**: Esercizi hands-on e notebook interattivi

## Prerequisiti

- Conoscenze base di programmazione (preferibilmente Python)
- Account su [Claude.ai](https://claude.ai) o accesso alle API Anthropic
- Curiosità e voglia di sperimentare!
