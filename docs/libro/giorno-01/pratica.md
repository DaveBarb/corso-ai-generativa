# Giorno 1: Pratica

## Esercizi del Giorno

La pratica del Giorno 1 è principalmente esplorativa e interattiva, svolta in classe con Claude.ai.

### Attività Suggerite

1. **Esplorazione di Claude.ai**
   - Creare un account su [claude.ai](https://claude.ai)
   - Familiarizzare con l'interfaccia
   - Provare diverse tipologie di prompt

2. **Demo Live**
   - Scrittura creativa: chiedere a Claude di scrivere una storia breve
   - Coding: generare una funzione Python con spiegazione
   - Analisi: caricare un grafico e farlo spiegare
   - Reasoning: problema logico risolto passo-passo

3. **Esperimenti con i Token**
   - Provare frasi in italiano vs inglese
   - Osservare come parole rare vengono tokenizzate
   - Capire il concetto di "context window"

### Domande di Riflessione Post-Pratica

1. Quali capacità dell'AI generativa ti sembrano più rilevanti per il tuo futuro professionale?

2. Quali rischi vedi nell'uso diffuso di queste tecnologie?

3. Se un LLM genera token uno alla volta, come fa a pianificare una risposta coerente?
