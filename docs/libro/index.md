# Introduzione al Corso

Benvenuti al corso di **AI Generativa** organizzato da ITS Steve Jobs.

## Panoramica

Questo corso di 40 ore è progettato per fornire una comprensione pratica e approfondita dell'Intelligenza Artificiale Generativa, con particolare focus su:

- **Fondamenti teorici** degli LLM (Large Language Models)
- **Prompt Engineering** avanzato
- **Sviluppo con API** Claude/Anthropic
- **Strumenti pratici** come Claude Code
- **Aspetti etici** e best practices
